Inherited vs. Non-Inherited Properties Inherited Properties:
Some CSS properties are automatically inherited by child elements, like:
  1. color
  2. font-family
  3. line-height
  4. text-align
Non-Inherited Properties: Other properties are not inherited by default, like:
  1. margin
  2. padding
  3. border
  4. background color
